Если следующий Java-код:

package tmp;

public class MainClass {
 public static void main(String[] args) {
  String str = "Hello World!";
  System.out.println(str);
 }
}

вставить в IDE Eclipse и запустить (горячие клавиши: Ctrl+F11) то во вкладке Output будет выведено: Hello World! (это значение берется из переменной str).
